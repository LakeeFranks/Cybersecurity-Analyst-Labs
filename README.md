# Splunk SIEM Labs (In Progress)

This repository documents my hands-on learning with **Splunk as a SIEM**, focused on understanding log ingestion, searching, and basic security analysis.

The goal of this lab is to build practical familiarity with how security-relevant events appear in logs and how they can be explored and analyzed in Splunk.

---

## 🧪 What I’m Working On
- Setting up a Splunk lab environment
- Ingesting and exploring **Windows Security logs**
- Analyzing authentication activity (successful and failed login attempts)
- Learning how to identify baseline behavior vs. suspicious patterns

---

## 📁 Repository Structure
- **Splunk Setup/**  
  Environment setup notes, configuration steps, and initial Splunk configuration

- **Windows Security Logs/**  
  Exploration of Windows authentication events, including failed and successful logins

---

## 🔧 Tools Used
- Splunk Enterprise
- Windows Event Logs (Security)
- Windows OS (VirtualBox VM)
